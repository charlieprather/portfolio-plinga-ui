Plinga Gaming UI Update

Problem:

The in-game "button bar" is old and is having a few problems. Each game on Plinga's platform has a different button bar design and this makes things complicated for our Devs, especially when we release a new game. The button bar is also not up to date with design trends regarding aesthetics and functionality.

Button bar to change:

Considerations

1.

We need to create something that represents Plinga
and unifies all of the games on the platform.

2.

The design is very old, it's time to update it and
utilize new design and functionality trends!

Brainstorming

While coming up with different ideas, I talked to people from different departments regarding current pain points and possible solutions. Some things that were talked about before any designing started were things like, will it be static on the page or will it push everything down when a button is clicked, like a classic hamburger menu.
